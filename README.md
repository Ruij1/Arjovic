## Overview
Web-based quotation platform built for Arjovic Muebles (arjovic.com.mx) to allow customers to request product quotes directly from the site, with automatic email delivery and backend storage using Google Sheets.

## Problem
The furniture store needed a dynamic and easy-to-use online system where customers could:
View products visually
Request quotes for selected items
Receive the quote via email
Store quote requests for business follow-up

## Solution
A responsive web platform built with:
HTML, CSS, and JavaScript for the interface
EmailJS for sending form data as email
Google Sheets as a lightweight backend to store and manage customer quote requests

## My Role
Front-end Developer & Integrator

## Key Features
Product catalog with images
Quotation form that sends requests via EmailJS
Integration with Google Sheets for centralized data storage
Interactive client experience (no backend server required)

## Tech Stack
HTML
CSS
JavaScript
EmailJS
Google Sheets API

## Impact
Customers can request quotes directly from the website
Store saves quote requests without manual entry
Replaced manual communication with automated workflow
